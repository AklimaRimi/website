# Image Recognizer 
This API can recognize insects namely: 
1. `Aphids`
2. `Fruit Fly`
3. `Caterpillar`
4. `Colorado Potato Beetle`
5. `Leaf Footed Bugs`
6. `Grasshoper`
7. `Thrips`
8. `Mexican Bean Beetle`
9. `Leaf Miners`
10. `White Flies`
11. `Mealy Bug`
12. `Spider Mites`
13. `Stink Bug`
